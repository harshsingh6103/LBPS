DTH with Wi-Fi Communication

System Overview

The DTH Monitoring System is an embedded solution that measures Dust, Humidity, and Temperature and displays the data via Wi-Fi. Measured values are stored in Excel format for reference and can be viewed remotely from any PC or mobile device through a provided link. The system automatically recovers and displays data after a power cut.

Key Features:
  • Remote Monitoring: Check the current status from anywhere using a provided link on a laptop, PC, or mobile device.
  • Alert System: If a value is out of range, it shows in RED on the display, and the corresponding sensor's LED will glow RED with a buzzer sound.
  • Data Logging: Measured data is stored in Excel format in a storage device.
  • Customizable Dashboard: The company name and logo can be displayed on the dashboard.
Standard Package:

Includes 4 Integrated Sensor units with adaptors and 1 Controller unit with a power cord. A display is not included.

DTH without Wi-Fi (Wired System)

System Overview

This version of the DTH Monitoring System provides a robust, high-security solution for local environmental monitoring. By using a direct wired connection from sensors to the controller, it eliminates reliance on Wi-Fi, making it ideal for environments with high RF interference or strict security protocols.

Key Features:
  • Local Monitoring: All data is displayed in real-time on a locally connected monitor or LCD screen.
  • Wired Reliability: Uses a stable wired connection (e.g., Ethernet or BGA Cable) for uninterrupted data transmission.
  • Enhanced Security: The closed-circuit nature of the system prevents wireless vulnerabilities.
  • On-Site Alerts: Features the same reliable color-coded alerts on the display and buzzer/LED alerts on the sensor units for immediate on-site awareness.
  • Internal Data Logging: All data is stored securely on the controller's internal memory, ready for manual export.
Standard Package:

Includes 4 wired Sensor units, 1 Controller unit, and required cabling. A display is not included.
