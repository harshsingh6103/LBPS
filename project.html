2 Channel Relay PCBA

Product Description

Our 2 Channel Relay PCBA is designed for basic control applications requiring reliable switching capabilities. This compact board features opto-isolation for enhanced safety and noise immunity.

Key Features

  • 2 SPDT relays (10A 250VAC / 10A 30VDC)
  • Opto-isolated inputs
  • LED status indicators for each channel
  • Screw terminal connections
  • Wide input voltage range (5-24V DC)
  • Compact DIN rail mountable design

Specifications

Relay Type Electromechanical SPDT
Contact Rating 10A @ 250VAC / 30VDC
Input Voltage 5-24V DC
Isolation Voltage 3750Vrms
Dimensions 85mm x 55mm x 25mm
4 Channel Relay PCBA

Product Description

The 4 Channel Relay PCBA offers enhanced protection circuits and supports both AC and DC loads. Ideal for industrial control applications requiring multiple switching points.

Key Features

  • 4 SPDT relays (10A 250VAC / 10A 30VDC)
  • Built-in flyback diodes for inductive loads
  • Opto-isolated inputs with status LEDs
  • DIN rail mounting capability
  • Wide operating temperature range (-20°C to 70°C)
  • Reverse polarity protection
