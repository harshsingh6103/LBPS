The machine operates via a simple and intuitive menu system.

1. Startup:
> LIGHT BROTHER > POWER SOLUTION
2. Access Settings:
> LBPS > Enter Pswd 00
3. Navigate Menu:
> Password Setup > Timer Setup
4. Door Status Alert:
> LBPS OPEN > 00:01 00:01
