# Light Brother Power Solutions (LBPS) - Official Website

![Website Status](www.lbpsindia.com)

This repository contains the source code for the official website of **Light Brother Power Solutions (LBPS)**, a leading manufacturer of specialized electronic solutions. The website is designed to showcase the company's products, mission, and expertise to clients and partners.

## 🚀 Live Demo

A live version of the website can be viewed here: **[www.lbpsindia.com](https://www.lbpsindia.com)** 

---

## ✨ Features

* **Responsive Design:** Fully responsive layout that works on all devices, from mobile phones to desktop computers.
* **Product Showcase:** Detailed pages for each product category, including:
    * Interface Devices
    * DTH Monitoring Systems
    * Thawing Machines
    * Temperature Monitors with Alarm
* **Interactive Modals:** Pop-up modals for viewing product details and requesting quotations.
* **Team & Company Info:** An "About Us" page detailing the company's mission and the expert team.
* **Functional Contact Form:** A working contact form powered by a backend service for inquiries.

---

## 🛠️ Technology Stack

This project is built with standard web technologies, ensuring high performance and maintainability.

* **Frontend:**
    * **HTML5**
    * **CSS3**
    * **Bootstrap 4:** For responsive layout and UI components.
    * **JavaScript (Vanilla & jQuery):** For DOM manipulation and interactivity.
* **Plugins:**
    * **MixItUp.js:** For filtering and sorting items in the product gallery.
