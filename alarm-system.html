Temperature Monitor Details

Key Features

  • Password Protected Menu: Prevents unauthorized changes to settings.
  • Configurable Temperature Range: Set custom minimum and maximum temperature limits.
  • Loud Audible Alarm: A buzzer sounds when temperature limits are breached.
  • Bright Visual Alarm: A flasher light provides a clear visual warning during an alarm state.
  • Manual Alarm Switch: Allows the user to silence the alarm when needed during an out-of-range event.
  • Automatic Reset: The alarm turns off automatically once the temperature returns to the safe range (Max Temp - 0.5°C).
